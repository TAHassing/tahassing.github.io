Milou van der Horst

Together with children's books writer Milou van der Horst I developed a website design to display her activities as a writer. I built a Rails 4 application using this design and implemented Active Admin as a content management system.

This web application offers the following features:

  • 'One-page' design with smooth scrolling and parallax effect on background
  • Fully dynamic content fields
  • A blog-like 'personal diary' system to communicate recent news in a playful manner
  • Book library, horizontal scroller, displays book details through in-site pop-ups

See it live: www.milouvanderhorst.nl (Dutch website)

JRecordings

Together with sound recording artist Jornt van Zandbergen I developed a website re-design to better suit his needs. I built a Rails 4 application using this design and implemented Active Admin as a content management system.

This web app offers the following features:

  • Site landing and contact pages built around photograph slider
  • Track record, listing musicians Jornt worked with, displays their details through in-site pop-ups
  • Multi-language (i18n) support for relevant parts

See it live: www.jrecordings.com

Wageningen45 Conflict Portal

For Wageningen45 I'm developing a Rails 4 web application that functions as an information hub on wars and conflicts in the world. Active Admin and front-end field edits are used for content management.

This web app offers the following features:

  • Article database, containing (links to) scientific articles and its authors
    related to conflicts, its causes, management thereof, etc.
  • Recent news section, displaying latest happenings regarding (global) conflicts
  • Calendar section, displaying events relevant to conflict managers
    on international scale such as conferences, symposia
  • Integrated map of current conflicts (developed by the UCDP program)
  • Discussion forum for users
  • User authentication through LinkedIn
  • Site moderation facilitated through different user levels and permissions
  • Multi-language (i18n) implemented throughout application
